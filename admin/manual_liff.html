LINE連携・LIFF実装マニュアル

応募フォームとLINEアカウントを連携し、LINE User IDを取得するための実装ガイドです。

概要・目的

ユーザーが応募フォームから登録を行う際、同時にLINE公式アカウントとの連携（LINE User IDの取得）を行う手法について解説します。
LIFF (LINE Front-end Framework) を使用することで、ユーザーは個別にIDを入力することなく、安全かつ確実にシステムと連携できます。

期待される効果

  • 応募と同時にLINE User IDがデータベースに保存される。
  • 「同姓同名」の問題を回避し、確実に本人に対してLINEメッセージを送信できる。
  • ユーザーの手間（ID手入力など）を最小限に抑えられる。

実装手順

既存の register.html をLIFFアプリとして動作させるための変更点です。

1. LINE Developers設定

設定項目 値・説明
LIFFアプリ名 例: プレミアム・イベント応募フォーム
サイズ Full (全画面) 推奨
エンドポイントURL https://(ドメイン)/register.html
※必ずHTTPSである必要があります
Scopes profile, openid

2. HTML/JSの改修

register.html に以下のコードを追加します。

<!-- 1. ヘッド内でLIFF SDKを読み込み --> <script charset="utf-8" src="https://static.line-scdn.net/liff/edge/2/sdk.js"></script> <!-- 2. フォーム内に隠しフィールドを追加 --> <form id="registrationForm"> ... <input type="hidden" id="line_user_id" name="line_user_id" value=""> ... </form> <!-- 3. JavaScriptで初期化とID取得 --> <script> document.addEventListener('DOMContentLoaded', async () => { try { // LINE Developersで取得したLIFF IDを設定 await liff.init({ liffId: "YOUR_LIFF_ID" }); // ログイン済み（LINEアプリ内）ならIDを取得 if (liff.isLoggedIn()) { const profile = await liff.getProfile(); const userId = profile.userId; // 隠しフィールドにセット document.getElementById('line_user_id').value = userId; console.log("LINE連携成功:", userId); } else { // 外部ブラウザの場合、必要ならログインを促す // liff.login(); } } catch (err) { console.error("LIFF初期化エラー", err); } }); </script>
