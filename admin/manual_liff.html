LINE連携・LIFF実装マニュアル

応募フォームとLINEアカウントを連携し、LINE User IDを取得するための実装ガイドです。

1. 概要・目的

ユーザーが応募フォームから登録を行う際、同時にLINE公式アカウントとの連携（LINE User IDの取得）を行う手法について解説します。
LIFF (LINE Front-end Framework) を使用することで、ユーザーは個別にIDを入力することなく、安全かつ確実にシステムと連携できます。

期待される効果

  • 応募と同時に LINE User ID がデータベースの line_user_id カラムに保存されます。
  • 「同姓同名」の問題を回避し、確実に本人に対してLINEメッセージを送信できます。
  • 将来的に、このIDを使って合否通知などの個別メッセージや一斉配信が可能になります。

2. LINE Developers設定

Messaging APIチャネルとは別に、LINEログインチャネルを作成する必要があります。

手順

  1. LINE Developers Console で「新規チャネル作成」を選択。
  2. チャネルタイプとして LINEログイン を選択（Messaging APIではありません）。
  3. 作成後、LIFF タブからLIFFアプリを追加。
  4. サイズ: Full を選択。
  5. エンドポイントURL: 本番環境の応募フォームURL（例: https://.../register.html）。
  6. Scope: openid と profile にチェック。
  7. 保存して発行された LIFF ID を控えます。

3. 実装コード

        document.getElementById('line_user_id').value = profile.userId;
